# Project Images Directory

## Folder Path
```
public/images/projects/
```

## Required Images

Place the following images in this folder:

1. **fire-alarm-installation.jpg** - Fire alarm installation project image
2. **hospital-fire-solutions.jpg** - Hospital fire solutions project image
3. **industrial-fire-safety.jpg** - Industrial fire safety project image
4. **data-center-protection.jpg** - Data center fire protection project image
5. **smart-fire-alarm.jpg** - Smart fire alarm system project image
6. **retail-fire-safety.jpg** - Retail fire safety project image

## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Recommended Size**: 800x600px or larger (aspect ratio: 4:3 or 16:9)
- **File Size**: Optimize images to keep file size under 500KB for better performance
- **Naming**: Use lowercase with hyphens (kebab-case) as shown above

## Image Paths in Code

The images are referenced in `src/components/sections/Projects.tsx` as:
- `/images/projects/fire-alarm-installation.jpg`
- `/images/projects/hospital-fire-solutions.jpg`
- `/images/projects/industrial-fire-safety.jpg`
- `/images/projects/data-center-protection.jpg`
- `/images/projects/smart-fire-alarm.jpg`
- `/images/projects/retail-fire-safety.jpg`

## Fallback

If an image is missing or fails to load, the card will automatically display a beautiful gradient background with the project icon.
