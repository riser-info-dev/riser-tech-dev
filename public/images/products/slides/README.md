# Cone Slide Design Images

This folder contains images for the cone/slide design component used in the "OUR FULL SPECTRUM FIRE SOLUTIONS" section.

## Folder Structure

```
public/images/products/slides/
├── slide-1.jpg
├── slide-2.jpg
├── slide-3.jpg
├── slide-4.jpg
└── slide-5.jpg
```

## Image Guidelines

1. **Image Format**: Use JPG or PNG format
2. **Image Size**: Recommended size is 800x1000 pixels (4:5 aspect ratio) for optimal display in the cone design
3. **File Naming**: Use `slide-1.jpg`, `slide-2.jpg`, etc.
4. **Content**: Images should showcase fire safety products, firefighting scenarios, or related fire safety solutions
5. **Optimization**: Optimize images for web to ensure fast loading
6. **Quality**: Use high-quality images that clearly represent your fire safety solutions

## Usage

The cone slide design will automatically cycle through all images in the slides folder. You can add more images by:
1. Adding the image file to this folder
2. Updating the image array in `app/products/page.tsx` to include the new image path

- Fire suppression systems
